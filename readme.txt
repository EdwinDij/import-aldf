---------------------------------------------
Générateur Word depuis Excel - Mode d'emploi
---------------------------------------------

Description :
Cette application permet de générer automatiquement des documents Word personnalisés
à partir d'un fichier Excel et d'un modèle Word (.docx).

Installation :
- Téléchargez et installez l'application.
- Sur macOS, glissez l'application dans le dossier Applications.
- Double-cliquez pour lancer l'application.

Utilisation :
1. Cliquez sur "Importer un fichier Excel" pour choisir votre fichier Excel (.xlsx).
2. Cliquez sur "Importer un modèle Word" pour sélectionner un fichier Word modèle (.docx).
3. Cliquez sur "Lancer le traitement" pour générer les documents.
4. Une barre de progression indique l'avancement.
5. Une fois terminé, cliquez sur "Ouvrir le dossier généré" pour accéder aux fichiers créés.
